# CONTRIBUTING

## Cómo contribuir
1. Trabaja sobre UTF-8.
2. Revisa primero `STYLEGUIDE.md`, `SCHEMA.md`, `EDITORIAL_GUIDE.md` y `ACCESSIBILITY.md`.
3. Mantén la arquitectura standalone si modificas HTML finales.
4. Si introduces nuevos datos, intenta respetar el super-schema base.

## Convenciones
- Usa nombres estables y consistentes.
- No mezcles variantes editoriales y técnicas sin documentarlo.
- Elimina archivos espurios antes de confirmar cambios (`Thumbs.db`, temporales, duplicados ambiguos).

## Antes de cerrar una contribución
- Ejecuta búsqueda de errores ortográficos obvios.
- Revisa acentos y diacríticos.
- Comprueba accesibilidad mínima.
- Documenta cualquier nueva convención o excepción.
