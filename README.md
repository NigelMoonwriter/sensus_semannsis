## Sobre el proyecto
Sensus Sēmannsis forma parte del Entorno Greka, una familia de diccionarios standalone que exploran el lenguaje desde diferentes ontologías:

SENSUS LATINUS — La memoria grecolatina. Etimología como arqueología del pensamiento.

SENSUS GRAECUS — Los fundamentos del pensamiento occidental. Logos, physis, ethos.

SENSUS SEMANNSIS — Reflexión sobre IA y creatividad. La máquina como espejo sin compasión.

SENSUS NEPANTLA — La frontera lingüística. El umbral donde el lenguaje se quiebra y renace.

Cada uno es un archivo independiente. Un solo HTML. Sin servidor. Sin base de datos externa. Sin tracking.

La familia completa puede vivir en cualquier dispositivo, funcionar sin internet, circular libremente.

Esa autonomía técnica es también una posición: el conocimiento que produce el Entorno Greka no necesita plataforma para existir.
