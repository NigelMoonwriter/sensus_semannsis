# STYLEGUIDE

## Nombres oficiales del ecosistema
- **ENTORNO GREKA**: nombre del ecosistema general.
- **SENSUS LATINUS**
- **SENSUS GRAECUS**
- **SENSUS NEPANTLA**
- **SENSUS SEMANNSIS**: forma editorial base en ASCII.
- **SENSUS SĒMANNSIS**: forma display permitida cuando se quiera conservar el macrón.

## Regla de nomenclatura
- En nombres de repositorio, rutas, slugs y claves técnicas: usar ASCII estable cuando sea posible.
- En títulos visibles, headings, portada y material editorial: se permiten diacríticos correctos.
- No mezclar variantes sin criterio dentro del mismo archivo.

## Acentos y diacríticos
- Escribir siempre en **UTF-8**.
- Corregir tildes en español: `básico`, `híbrido`, `máquina`, `tránsito`, `búsqueda`, `ficción`.
- No usar variantes erróneas como `basíco`, `hibrido` o `maquina` salvo que formen parte de una cita histórica exacta.
- Para griego transliterado, conservar una convención estable por archivo.

## Capitalización
- Títulos de obra/diccionario: mayúsculas sostenidas permitidas.
- Cuerpo de texto: capitalización normal del español.
- Campos de datos: minúsculas consistentes (`norte`, `sur`, `este`, `oeste`).

## Hashtags y labels visibles
- Usar `#ReflejosHíbridos` como forma editorial preferente.
- Si una plataforma no soporta diacríticos o rompe el enlace, documentar explícitamente la variante ASCII usada.

## Interfaz y microcopy
- El texto de UI debe estar en español salvo razón editorial clara.
- Botones, filtros y mensajes vacíos deben llevar ortografía correcta.
- Evitar mezclar tono técnico y poético dentro del mismo control si afecta claridad.
